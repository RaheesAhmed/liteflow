# Getting Started with LiteFlow

Welcome to LiteFlow! This guide will help you get started with building modern web applications using LiteFlow.

## Quick Start

Create a new LiteFlow project with a single command:

```bash
npx create-liteflow-app my-app
cd my-app
npm run dev
```

## What's Included

Your new LiteFlow project comes with:

- ⚡️ Instant HMR and development server
- 🔒 TypeScript by default
- 🎨 Tailwind CSS and shadcn/ui components
- 🔐 Authentication ready
- 📦 Database with Prisma
- 🚀 API routes with validation
- 📱 Responsive layouts
- 🌙 Dark mode support

## Key Features

### 1. File-Based Routing

LiteFlow uses an intuitive file-based routing system:

```tsx
// app/posts/[id]/page.tsx
export default function PostPage({ params }: { params: { id: string } }) {
  return <h1>Post {params.id}</h1>;
}
```

### 2. Data Fetching

Simple and powerful data fetching:

```tsx
import { prisma } from "@liteflow/db";

// Server Component
export default async function PostList() {
  const posts = await prisma.post.findMany();
  return <div>{/* Render posts */}</div>;
}
```

### 3. API Routes

Type-safe API routes with validation:

```tsx
import { z } from "zod";
import { auth } from "@liteflow/auth";
import { prisma } from "@liteflow/db";

const schema = z.object({
  title: z.string(),
  content: z.string(),
});

export async function POST(request: Request) {
  const session = await auth();
  const body = await request.json();
  const data = schema.parse(body);

  const post = await prisma.post.create({
    data: {
      ...data,
      authorId: session.user.id,
    },
  });

  return Response.json(post);
}
```

### 4. Authentication

Built-in authentication support:

```tsx
import { auth } from "@liteflow/auth";

export default async function Profile() {
  const session = await auth();

  if (!session) {
    return <div>Please sign in</div>;
  }

  return <div>Welcome, {session.user.name}</div>;
}
```
